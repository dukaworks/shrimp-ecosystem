## 🎯 生态愿景

**虾族生态 (Shrimp Clan Ecosystem)** 是一套围绕 OpenClaw 构建的专业工具集，
致力于让 OpenClaw 的部署、使用、维护、迁移变得简单、安全、可逆。

### 核心理念

```
🔄 闭环生态
部署 → 使用 → 备份 → 升级/迁移 → 恢复 → 重新部署
     ↑___________________________________________|

🛡️ 安全第一
任何操作都可逆，任何配置都可备份

🎨 体验至上
可爱的 CLI 界面，让技术工作也有温度
```

## 🎯 快速开始（5分钟上手）

### 第一步：安装三叉戟

```bash
# 一键安装所有工具
pip install openclaw-feishu-deployer openclaw-uninstaller openclaw-snapshot

# 验证安装
ofd --help    # 部署工具
ocu --help    # 卸载工具  
ocs --help    # 快照工具
```

### 第二步：部署 OpenClaw

```bash
# 启动部署向导
ofd deploy

# 按提示完成：
# 1. 选择模型（Kimi/GPT/Claude）
# 2. 配置飞书机器人
# 3. 完成部署
```

### 第三步：创建初始快照

```bash
# 保存纯净安装状态
ocs fresh

# 输入名称: fresh_install
# 类型: 纯净安装
```

### 第四步：日常使用

```bash
# 每周备份
ocs create
# 名称: weekly_$(date +%Y%m%d)
# 类型: 当前状态
```

### 第五步（如需要）：安全卸载

```bash
# 卸载前自动备份
ocu

# 选择: 📦 保存为 tar.gz
# 保存路径: ~/Desktop/
```
