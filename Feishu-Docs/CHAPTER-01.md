## 1.1 什么是虾族生态？

虾族生态是一套围绕 **OpenClaw** 构建的专业工具集，提供从部署、使用、备份到卸载的全生命周期管理。

### 🎯 我们的使命

让每个人都能轻松管理 OpenClaw，无论是新手还是专家。

### 💡 核心理念

【信息框 - 蓝色】
📌 **闭环生态**
部署 → 使用 → 备份 → 升级/迁移 → 恢复 → 重新部署

🛡️ **安全第一**
任何操作都可逆，任何配置都可备份

🎨 **体验至上**
可爱的 CLI 界面，让技术工作也有温度

---

## 1.2 三叉戟工具集

虾族生态目前包含三个核心工具：

【表格】
| 工具 | 图标 | 功能 | 命令 | 适用场景 |
|------|------|------|------|----------|
| 🦞 部署虾 | 🦞 | OpenClaw + 飞书一键部署 | `ofd` | 新用户安装 |
| 💾 备份虾 | 💾 | 智能快照管理 | `ocs` | 日常备份、迁移 |
| 💥 卸载虾 | 💥 | 完全卸载 + 自动快照 | `ocu` | 重装、清理 |

## 1.3 5分钟快速体验

### 安装（1分钟）

【代码块 - bash】
```bash
pip install openclaw-feishu-deployer openclaw-uninstaller openclaw-snapshot
```

【信息框 - 绿色】
✅ 提示：需要 Python 3.7+ 和 Node.js 18+

### 部署（3分钟）

【代码块 - bash】
```bash
ofd deploy
```

按提示完成：
1. 选择 AI 模型（Kimi/GPT/Claude）
2. 配置飞书机器人
3. 完成部署

### 备份（1分钟）

【代码块 - bash】
```bash
ocs fresh
```

输入名称：`my_first_backup`
选择类型：`纯净安装`

【信息框 - 粉色】
🎉 恭喜！你已经完成了虾族生态的初次体验！

## 1.4 为什么选择虾族？

【折叠块 - 点击查看详情】
### vs 手动安装

| 方式 | 时间 | 难度 | 可恢复性 |
|------|------|------|----------|
| 手动安装 | 30-60分钟 | 高 | 低 |
| 虾族生态 | 5分钟 | 低 | 高（快照） |

### vs 其他工具

- **一体化**：部署 + 备份 + 卸载，完整闭环
- **可爱界面**：不再是枯燥的命令行
- **安全可靠**：任何操作都可逆
- **社区支持**：持续更新和维护
