## 页面设计规范

### 视觉风格

```yaml
主色调:
  粉色: #FF6B9D    # 用于强调、按钮
  青色: #4ECDC4    # 用于成功、提示
  黄色: #FFE66D    # 用于警告、注意
  背景: #FAFAFA    # 页面背景

字体:
  标题: 思源黑体 Bold, 24px
  正文: 思源黑体 Regular, 14px
  代码: JetBrains Mono, 13px

图标:
  使用 Emoji + 自定义图标
  每个工具一个专属图标
```

```markdown
1. 【信息框】用于提示

---

## 步骤 1：安装部署工具

### 🎯 目标
安装 `openclaw-feishu-deployer` 工具

### 💻 操作

```bash
pip install openclaw-feishu-deployer
```

### ✅ 验证

```bash
ofd --version
# 应显示版本号
```

### 📖 详细说明
> 【折叠块：pip 安装常见问题】
> 
> **Q: pip 命令找不到？**
> ```bash
> # macOS
> brew install python
> 
> # Ubuntu
> sudo apt install python3-pip
> ```
> 
> **Q: 权限不足？**
> ```bash
> pip install --user openclaw-feishu-deployer
> ```
